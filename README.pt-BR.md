# Superstore Sales Analytics
English version below  
Versão em português abaixo


This project analyzes the Superstore dataset using **Power BI and Python** to explore sales performance, profitability, and business insights.

The objective is to demonstrate a complete data analytics workflow, including exploratory data analysis, trend modeling, and sales forecasting.

---

## Project Structure

superstore-analytics

- superstore_analysis.ipynb
-  powerbi_dashboard.pbix
-   Sample - Superstore.csv
-  README.md


---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Power BI

---

## Analysis Performed

### 1. Data Inspection
- Dataset structure
- Summary statistics
- Data types

### 2. Exploratory Data Analysis
- Sales by category
- Sales distribution

### 3. Correlation Analysis
Analyzed relationships between:

- Sales
- Profit
- Discount
- Quantity

### 4. Sales Trend Analysis
Aggregated sales by month to identify seasonality and growth patterns.

### 5. Linear Regression Trend
A linear regression model was applied to identify the overall sales trend.

Model performance was evaluated using **R² score**.

### 6. Impact of Discount on Profit
Regression analysis shows the relationship between discount levels and profitability.

### 7. Product Performance Analysis
Scatter plot comparing **Revenue vs Profit by product**.

### 8. Profit Margin by Category
Identifies which product categories generate the highest margins.

### 9. Negative Profit Orders
Analysis of orders generating losses and identification of products responsible for the largest losses.

### 10. Sales Forecast
A basic forecast was generated using the regression model to estimate sales for the next 6 months.

---

## Key Business Insights

- Sales increase toward the end of the year, indicating seasonal demand.
- The Technology category generates the highest revenue.
- High discount levels significantly reduce profitability.
- Some products generate losses due to aggressive discount strategies.
- Monitoring discount policies can improve profit margins.

A Power BI dashboard was developed to complement the analysis with interactive visualizations including:

- Sales Performance Overview
- Product Performance Analysis

# Análise de Vendas - Superstore (PT)

Este projeto analisa o dataset Superstore utilizando Power BI e Python para explorar desempenho de vendas, lucratividade e gerar insights de negócio.

O objetivo é demonstrar um fluxo completo de análise de dados, incluindo análise exploratória, modelagem de tendência e previsão de vendas.

## Análises Realizadas

### 1. Inspeção dos Dados
- Estrutura do dataset
- Estatísticas descritivas
- Tipos de dados

### 2. Análise Exploratória
- Vendas por categoria
- Distribuição de vendas

### 3. Análise de Correlação
Relação entre as variáveis:

- Sales
- Profit
- Discount
- Quantity

### 4. Tendência de Vendas
Agregação das vendas por mês para identificar sazonalidade e padrões de crescimento.

### 5. Regressão Linear
Aplicação de modelo de regressão linear para identificar a tendência geral de vendas.

O modelo foi avaliado utilizando **R²**.

### 6. Impacto de Desconto no Lucro
Análise da relação entre nível de desconto e lucratividade.

### 7. Análise de Produtos
Comparação entre **Receita e Lucro por produto**.

### 8. Margem de Lucro por Categoria
Identificação das categorias com maior margem.

### 9. Pedidos com Prejuízo
Identificação de pedidos que geraram prejuízo e produtos responsáveis pelas maiores perdas.

### 10. Previsão de Vendas
Previsão simples para os próximos 6 meses utilizando regressão linear.

---

## Insights Principais

- As vendas aumentam no final do ano indicando sazonalidade.
- A categoria Technologia gera maior receita.
- Descontos elevados reduzem significativamente a lucratividade.
- Alguns produtos geram prejuízo devido a estratégias agressivas de desconto.
- O monitoramento de políticas de desconto pode melhorar margens.
