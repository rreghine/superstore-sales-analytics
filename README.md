# Superstore Sales Analytics

This project analyzes the Superstore dataset using **Power BI and Python** to explore sales performance, profitability, and business insights.

The objective is to demonstrate a complete data analytics workflow, including exploratory data analysis, trend modeling, and sales forecasting.

## Analysis Performed

### 1. Data Inspection
- Dataset structure
- Summary statistics
- Data types

### 2. Exploratory Data Analysis
- Sales by category
- Sales distribution

### 3. Correlation Analysis
Analyzed relationships between:

- Sales
- Profit
- Discount
- Quantity

### 4. Sales Trend Analysis
Aggregated sales by month to identify seasonality and growth patterns.

### 5. Linear Regression Trend
A linear regression model was applied to identify the overall sales trend.

Model performance was evaluated using the **RÂ˛ score**.

### 6. Impact of Discount on Profit
Regression analysis shows the relationship between discount levels and profitability.

### 7. Product Performance Analysis
Scatter plot comparing **Revenue vs Profit by product**.

### 8. Profit Margin by Category
Identifies which product categories generate the highest margins.

### 9. Negative Profit Orders
Analysis of orders generating losses and identification of products responsible for the largest losses.

### 10. Sales Forecast
A basic forecast was generated using the regression model to estimate sales for the next 6 months.

---

## Key Business Insights

- Sales increase toward the end of the year, indicating seasonal demand.
- The Technology category generates the highest revenue.
- High discount levels significantly reduce profitability.
- Some products generate losses due to aggressive discount strategies.
- Monitoring discount policies can improve profit margins.
